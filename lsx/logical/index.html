__m128i __lsx_vand_v (__m128i a, __m128i b)

Synopsis

__m128i __lsx_vand_v (__m128i a, __m128i b)
#include <lsxintrin.h>
Instruction: vand.v vr, vr, vr
CPU Flags: LSX

Description

Compute bitwise AND between elements in a and b.

Operation

for (int i = 0; i < 2; i++) {
  dst.dword[i] = a.dword[i] & b.dword[i];
}

Tested on real machine.

Latency and Throughput
